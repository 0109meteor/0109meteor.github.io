Navigation

pwd

pwd: 看目前位置(print working directory)

用GUI打開路徑

open <路徑> : 用finder打開某個路徑（適用mac）

explorer.exe <路徑>: 用檔案總管打開路徑（適用WSL）

  • . 代表目前路徑

ls

ls: 列出目前位置檔案(list)

  • -a: 顯示隱藏檔案
  • -l: 顯示詳細資訊
  • -la:顯示隱藏檔案&詳細資訊
  • ls <路徑> : 後面可加上路徑，就可以列出某個路徑下的檔案

cd

cd <路徑>: 移動到其他位置(change directory)

  • ..代表上一層路徑

man

man <指令>: 查看man page（說明文件）

相對路徑 v.s. 絕對路徑

  • 絕對路徑：/Users/jamesku/repos
  • 如果你現在在/Users/jamesku/ 上面的路徑轉換成相對路徑就是repos

File

mkdir

mkdir <資料夾路徑>: 建立資料夾

cat

cat: 印出檔案內容

cp

cp <來源> <目的>: 複製檔案

  • -r: recursive（複製資料夾時可使用）

rm

rm <路徑>: 刪除檔案

  • -r: recursive（刪除資料夾時可使用）

mv

mv <來源> <目的>: 移動檔案

  • 可用於重新命名

vim

vim <檔案名稱>：用vim開啟檔案

  • 可以打開不存在的檔名，存檔後會出現新的檔案
  • vim教學影片

雜項

上下

只要按上跟下就可以選擇之前用過的指令，不用重打一次

control + R

如果你想要找很久以前輸入的指令，可以按下control + R，然後輸入文字，就可以搜尋了。如果找到的不是你要的，可以繼續案control + R繼續往前搜尋。

control + S

如果想要往後搜尋，則是按下control + S

package manager

package manager就是拿來安裝東西的工具，就像手機的apple store，或是play store一樣，不過要使用指令來操作。不管是用哪種系統，package manager都是非常常用的工具，所以一定要知道怎麼使用。

舉幾個例子，如果是mac電腦，通常會使用brew這個指令來安裝東西，ubuntu的話＄就會使用apt這個指令，manjaro則是使用pacman。
